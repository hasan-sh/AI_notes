> Subject: Machine learning: Intro & trends.

Hoogedoorn's position is ML and he loves it!

Defs:

Hype cycle: a representation of new emerging techs(see the pic on google)

  • innovation triggers; where the start is.
  • ML won't be the solution for everything in the world.

ML:


  • is to automatically identify patterns from data
  • "a computer program is said to learn from experience E with respect to some class of tasks T and performance P, if its performance at tasks in T improves with E" ~mitchell

why is ML becoming popular?


  • increasing amount of data available
  • more computational power
  • better algorithms; to exploit the data in better way

Tasks solved by ML:


  1. supervised learning: is the ml task of inferring a function from a set of labeled training

  • Image dataset: medical problem: the dataset is consist of rows and columns, which represent patients and their medical situation; each of which has features and target whether they're applicable or not; based on what the ML's process provide; e.g. whether their risk is high, low or medium; > for each of node see if a person has problems with feature 1 if higher than 2 then if they have problems with features 3 if lower make their label as yes. >it's using a tree search algorithm to know which person's health is low, high or medium.
    • neural networks:
> many hidden layers then comes the classification process.

  • evaluation:
    • want to have models predict well for new examples
    • train them(ml) on a training set and apply our algorithm to an independent test set to assess generalizability.
    • this will show us how good it will work on new examples.
> i.g. 1000 patients select a bunch of them and train with the decision tree take the rest and train them again and then comapre how th ml process did and the percentage it gave for each.


> the data provided should be good to understand the problem and tackle it.
