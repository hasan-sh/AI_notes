float types stored in memory are not exact; if you got a huge float number doing calculation could result in errors.

  • Fixed point notation => 1.3 or 1.0002
  • Floating point notation => for example the number representing the charge of an electron 1.6 * 10^-19
    • In python this would be: 1.6e-19

Sequence types:

  • str: a row of a character; symbol
  • list: a row of anything; string, number or list


This lecture was a very basic introduction to Python and mainly about its syntax, primitive and non-primitive types, and how to use each type.
