Schnapsen | Practical part in the course


Topics of importance

  • Representing as a state-space
  • Searching for (optimal) solutions
  • Multi-agent behaviour
  • Learning strategies and heuristics
  • Representing strategies as logical rules

  • First week we'll look into how the machine makes decisions when perfect information is provided
  • Second week the same. but when partial information provided.


Goals

  1. basic knowledge of what agents and algorithms are.
  2. get familiar with the code of schnapsen
  3. week 5: implementing own intelligent agent that applies reasoning to make implicit facts explicit!



Grades

  • exam = 75%
  • practical = 25%
    • at least 3 bots(practical assignments)
      • min-max = 10 points
      • adaptive = 10 points
      • using knowledge = 5 points
Make an appendix in which you explain how you implemented your code and the rationale behind it.
